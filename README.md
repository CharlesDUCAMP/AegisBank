# **Aegis Bank — Compte Créateurs**

**Landing page** réalisée dans le cadre du projet web (**Q1**).

---

### **Objectif**
Créer une interface **mobile-first** pour une néobanque éthique destinée aux **créateurs de contenu**.

### **Fonctionnalités**
* **Navigation responsive :** Menu burger animé (transformation en "X") avec dropdown latéral.
* **Concept "Zero-Scroll" :** Optimisation du viewport pour afficher l'essentiel sans défilement.
* **Design System :** Esthétique typée **"Twitch/Gaming"** (Dark mode et accents violets).
* **Animations fluides :** Effets de lueur (**glow**) et micro-interactions au survol.

### **Technologies**
* **HTML5** (Structure sémantique)
* **CSS3** (Grid & Flexbox pour le centrage absolu, Keyframes pour les animations)
